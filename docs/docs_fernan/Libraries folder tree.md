# **_Aixt_** Libraries folder tree

Libraries in **_Aixt_** are located in the `lib` folder following this folder tree:

```
aixt/lib/
    ├── arduino/
    │   ├── arduino_lib/
    │   │   ├── arduino_lib.c
    │   │   ├── arduino_lib.c.v
    │   │   └── function_1/
    │   │       ├── function_1.c
    │   │       └── function_1.c.v
    :   :
    :
    ├── c/
    │   ├── c_lib/
    │   │   ├── c_lib.c
    │   │   ├── c_lib.c.v
    │   │   └── function_1/
    │   │       ├── function_1.c
    │   │       └── function_1.c.v
    :   :
    :
    ├── nxc/
    │   ├── nxc_lib/
    │   │   ├── nxc_lib.c
    │   │   ├── nxc_lib.c.v
    │   │   └── function_1/
    │   │       ├── function_1.c
    │   │       └── function_1.c.v
    :   :
    :
    ├── generic_lib/
    │   ├── generic_lib.c
    │   ├── generic_lib.c.v
    │   └── function_1/
    │       ├── function_1.c
    │       └── function_1.c.v
    :
```
